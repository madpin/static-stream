Publish Static Camera

  1. Join the Apple Developer Program and add the account in Xcode Settings > Accounts.
  2. Create an Apple Development certificate from Manage Certificates.
  3. Enable System Extension for the host identifier and App Groups for both identifiers.
  4. Build with SIGNING_IDENTITY="Apple Development: Name (TEAMID)" TEAM_IDENTIFIER_PREFIX="TEAMID" ./scripts/build-macos.sh.
  5. Move Static Stream.app to /Applications, choose Install / update, and approve Camera Extensions in System Settings.
  6. Select Static Camera in the meeting app, then use Live or Freeze in Static Stream.
